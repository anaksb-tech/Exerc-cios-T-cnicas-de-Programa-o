O imperador romano Júlio César criptografava mensagens trocando cada letra pela
terceira letra seguinte no alfabeto (ex: 'a' vira 'd', 'b' vira 'e', e assim
por diante).

Crie uma classe chamada Cifra, sem atributos, contendo dois métodos:
- criptografar(String mensagem): recebe uma string e retorna a versão cifrada
- descriptografar(String cifrada): recebe a string cifrada e retorna a original

No método main, teste os dois métodos com alguma mensagem de sua escolha.
