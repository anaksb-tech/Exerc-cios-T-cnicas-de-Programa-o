Crie uma classe chamada Funcionario com três atributos: primeiro nome, sobrenome
e salário mensal. O salário deve ser um valor decimal entre 0 e 20000.

A classe deve ter um construtor que inicializa os três atributos. Caso nenhum
valor seja passado, use strings vazias para os nomes e 0 para o salário.

Implemente getters e setters onde necessário. O setter do salário deve ignorar
qualquer tentativa de atribuir um valor fora do intervalo permitido, mantendo
o valor anterior.

No método main, crie duas instâncias de Funcionario, exiba seus nomes e salários,
aplique um aumento de 20% e exiba novamente.
