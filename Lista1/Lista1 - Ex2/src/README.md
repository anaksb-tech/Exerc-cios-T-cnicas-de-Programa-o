Crie uma classe chamada Produto para representar um item de uma loja de materiais.
A classe deve ter quatro atributos: número de identificação (inteiro de 1 a 9999),
quantidade em estoque (inteiro não negativo), nome do produto (texto) e preço
unitário (decimal não negativo).

A classe deve ter um construtor que inicializa os quatro atributos de forma válida.
Implemente getters e setters onde necessário para garantir que valores inválidos
não sejam aceitos.

Valores padrão: id = 1, quantidade = 0, preço = 0, nome = "".
