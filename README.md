# Exercícios de Técnicas de Programação

Este repositório armazena as soluções para as listas de exercícios propostas durante a disciplina de Técnicas de Programação. As implementações estão em Java e os arquivos contêm o código-fonte (.java) de cada solução.

### Conteúdo do Repositório
As listas de exercícios estão organizadas por tópicos abordados na disciplina. Cada pasta contém o código-fonte relacionado aos conceitos aplicados.

Lista 1 - Encapsulamento, Construtores e Vetores: Exercícios focados na proteção de dados (encapsulamento), inicialização de objetos (construtores) e manipulação básica de arrays (vetores).

Lista 2 - Strings e Vetores: Práticas envolvendo a manipulação e formatação de Strings, além de operações e lógicas mais complexas utilizando vetores.

Lista 3 - Herança: Implementação e exercícios explorando o conceito de herança, reaproveitamento de código e hierarquia de classes na Programação Orientada a Objetos.
