Crie uma classe chamada Tweet que possui como atributo um parágrafo de no
máximo 144 caracteres. Avalie se há necessidade de getters e setters e, se
houver, implemente-os com a visibilidade adequada.

A classe deve ter os seguintes métodos:
- contarCaracteres(): retorna a quantidade de caracteres do parágrafo
- contarPalavras(): retorna a quantidade de palavras do parágrafo
- contarFrases(): retorna a quantidade de frases do parágrafo
- contarOcorrencias(String palavra): retorna quantas vezes a palavra aparece
  no parágrafo
- gerarParagrafoAleatorio(): método que pode ser chamado sem instanciar um
  objeto da classe, retornando uma String gerada aleatoriamente
